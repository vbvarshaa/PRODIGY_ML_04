# PRODIGY_ML_04
Hand gesture recognition model

This project classifies different hand gestures using a machine learning model.

## Steps
- Loaded and preprocessed images (resize + flatten)
- Encoded labels
- Trained SVM model
- Evaluated accuracy

## Result
- Accuracy: ~99.6%
- Successfully predicts hand gestures

## Tools
Python, OpenCV, NumPy, Scikit-learn
